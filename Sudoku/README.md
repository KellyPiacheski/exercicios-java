# Exercícios Java 🖥️

Coleção de exercícios práticos desenvolvidos durante o curso de Software Developer (Desenvolvimento Web Full-Stack) da CESAE Digital — programa Reskilling 4 Employment.

## Exercícios

### 🎮 Sudoku
Programa interativo para manipular quadrículas Sudoku 9x9.

**Funcionalidades:**
- Geração automática de quadrícula inicial válida
- Permuta de dois números em toda a grelha
- Permuta de linhas e colunas dentro da mesma faixa
- Permuta de faixas horizontais e verticais
- Reflexão horizontal e vertical
- Introdução manual de nova quadrícula
- Validação de quadrícula (linhas, colunas e blocos 3x3)

**Como executar:**
```bash
javac Sudoku2.java
java Sudoku2
```
